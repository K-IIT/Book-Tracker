# Book Tracker

**Автор:** [Ваше Имя Фамилия]

## Описание программы

Book Tracker — простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Есть фильтрация по жанру и количеству страниц. Данные сохраняются в файл books.json.

## Инструкция по запуску

1. Установите Python 3.8+.
2. Клонируйте репозиторий:
   ```
   git clone <ссылка_на_репозиторий>
   ```
3. Перейдите в папку проекта:
   ```
   cd book_tracker
   ```
4. Запустите приложение:
   ```
   python book_tracker.py
   ```

## Возможности

- Добавление книг с проверкой ввода.
- Фильтрация по жанру и количеству страниц.
- Сохранение и загрузка данных из JSON.
