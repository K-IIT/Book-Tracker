# Book Tracker

**Автор:** Шкерина Дарья Олеговна
**Язык:** Python 3.8+
**Лицензия:** MIT (по желанию)

## Описание программы

Book Tracker — графическое приложение для учёта прочитанных книг.
Позволяет добавлять книги с проверкой данных (название, автор, жанр, количество страниц), фильтровать список по жанру и количеству страниц.
Все данные сохраняются в файл `books.json` в формате JSON.
Используется стандартная библиотека `tkinter` для интерфейса.
